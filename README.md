# WPF-Practise-VirtualFileSystemUI
 对 [c++练手 : 基于文件内存映射的虚拟文件系统] 使用WPF编写的UI  
 
>功能  
   1.支持单个/多个文件/文件夹导入/导出，支持导入时类型筛选  
   2.支持历史记录跳转  
   3.支持通过名称搜索文件/文件  
   4.支持重命名、新建分区、新建文件夹
